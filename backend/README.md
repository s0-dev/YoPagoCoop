# YoPagoCoop Backend

Este repositorio contiene la implementación del backend para el sistema de YoPagoCoop. El backend esta desarrollado usando Spring Boot y PostgreSQL, llevando a cabo un patron de diseño modular para manejar las escuelas y sus miembros con soporte para atributos especificos para cada uno.
